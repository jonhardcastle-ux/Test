# Code Standards

## Accessibility
- All projects must meet WCAG 2.1 AA
- Every image needs descriptive alt text
- Use semantic HTML (nav, main, aside, article)
- Ensure keyboard navigability on all interactive elements
- Minimum contrast ratio: 4.5:1

## Code Style
- TypeScript strict mode always on
- Functional React components only, no class components
- CSS: use BEM naming convention
- No inline styles

## Testing
- Every component needs a unit test
- Use React Testing Library, not Enzyme

## Tone of Voice
- Warmly Confident: The brand combines expert, evidence-based knowledge with a welcoming, accessible manner.
- Passionately Supportive: The voice is encouraging, focusing on building emotional well-being and supporting educators, leaders, and practitioners.
- Boldly Purposeful: The tone is direct and driven by a mission to improve learning and development through innovation and personalized experiences.
- Human & Engaging: Even while leveraging AI (Kiki), the tone aims to be human, intuitive, and engaging, moving away from robotic or overly formal language.
- Impact-Driven: The content is designed to be concise, actionable, and tailored to the modern learner, often mirroring the fast-paced, engaging style of platforms like TikTok or LinkedIn.
